# DevOps Go Project

## Run locally

docker-compose up --build

## Open
http://localhost:8080

## Features
- Go API
- Dockerized
- CI/CD with GitHub Actions
- Tests
- Auto deploy via SSH

## Setup CD

Add GitHub Secrets:
- HOST (server IP)
- USER (ssh user)
- SSH_KEY (private key)
